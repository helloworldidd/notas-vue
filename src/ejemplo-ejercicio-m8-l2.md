# Ejemplo M8 L2


---

## Objetivo

Revisar un proyecto desarrollado durante el curso, corregir errores, aplicar mejoras e incorporar el feedback recibido.

> **Puedes utilizar cualquier proyecto realizado durante el curso. No es obligatorio usar el proyecto del clima ni un proyecto desarrollado con Vue.**

---

<br>
<br>
<br>
<br>

## ¿Qué debes hacer?

<br>
<br>
<br>
<br>

---

## 1. Revisar el proyecto

Identifica oportunidades de mejora, por ejemplo:

* Errores o bugs
* Funcionalidades incompletas
* Código repetido
* Organización del proyecto
* Diseño y usabilidad
* Rendimiento

---

## 2. Aplicar mejoras

Realiza las correcciones necesarias, por ejemplo:

* Corregir errores
* Optimizar el código
* Mejorar la interfaz
* Reutilizar componentes o funciones
* Agregar validaciones
* Aplicar buenas prácticas

---

## 3. Incorporar feedback

Si recibiste observaciones durante el curso, intégralas al proyecto y explica brevemente cómo contribuyeron a mejorarlo.

---

## 4. Documentar los cambios

Entrega un documento (PDF o DOCX) indicando:

* Errores detectados y corregidos.
* Mejoras implementadas.
* Feedback recibido y aplicado.

---

## Entrega

* Proyecto actualizado (ZIP o enlace al repositorio).
* Documento PDF o DOCX con las mejoras realizadas.

---

## Importante

La evaluación se centra en tu capacidad para **revisar, mejorar y dejar una versión final de un proyecto existente**, demostrando un proceso de mejora continua.
