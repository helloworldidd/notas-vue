## 





1. presionar: **Ctrl + Shift + P**

2. eligir: **Snippets: Configure User Snippets**

3. eligir: **vue.json**

4. agregar:

```json
{
  "Vue 3 Base": {
    "prefix": "vue3",
    "body": [
      "<script setup>",
      "",
      "</script>",
      "",
      "<template>",
      "  <div>",
      "    $1",
      "  </div>",
      "</template>",
      "",
      "<style scoped>",
      "",
      "</style>"
    ],
    "description": "Componente base Vue 3"
  }
}
```

5. Guardar y ahora en archivos **.vue** puedo usar **vue3**
