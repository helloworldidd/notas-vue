

### CICLO DE VIDA



| Hook            | Momento               | Uso típico              |
| --------------- | --------------------- | ----------------------- |
| onBeforeMount   | Antes de mostrar      | Preparación             |
| onMounted       | Después de mostrar    | API, LocalStorage       |
| onBeforeUpdate  | Antes de actualizar   | Comparar valores        |
| onUpdated       | Después de actualizar | Revisar cambios         |
| onBeforeUnmount | Antes de destruir     | Limpiar eventos, timers |
| onUnmounted     | Después de destruir   | Confirmar limpieza      |
