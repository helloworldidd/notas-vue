# Deploy

## GitHub Pages

Sirve para publicar proyectos estáticos.

Bueno para:

```text
landing pages
portafolios
proyectos Vue simples
catálogos sin backend
```

---

## Vercel

Sirve para publicar proyectos modernos de frontend.

Bueno para:

```text
Vue
React
Nuxt
Next
sitios conectados a APIs
deploy automático desde GitHub
```

---

# Funciones más avanzadas

Para cosas como:

```text
transacciones
favoritos
guardar datos de usuarios
subir imágenes
chat o WhatsApp
formularios reales
```

Necesitas conectar un backend o servicio externo.

Opciones:

```text
Supabase
Firebase
WordPress API
Strapi
Sanity
```

# SEO

## Vue normal

Vue con Vite funciona muy bien, pero para SEO avanzado puede quedarse corto porque muchas cosas se cargan con JavaScript.

---

## Solución pro: Nuxt

Para SEO real, lo mejor es usar:

```text
Nuxt
```

Nuxt permite:

```text
SSR
SSG
mejor SEO
rutas automáticas
mejor rendimiento inicial
```
